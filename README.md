# thread-demo

```
Just make a collection of 20 items any data type. these 20 items represents 20 tasks. 
Have multiple Thread access, start with 2 threads.
Write multi-threaded solution, make these 2 threads busy to complete 20 tasks.  
Print entire status of tasks and threads allocation.

No task should repeat by any thread, No task should be missed. 
Put Dummy Processing for 1 second in worker thread. Make parallel worker threads. Use wait, notify as required.
```
